# Signal Hunters Web

Landing page static cho Signal Hunters với phong cách dark editorial chuyên nghiệp, không neon, xây bằng HTML/CSS/JS thuần.

## Điểm chính

- Header sticky với CTA Telegram
- Ticker bar chạy marquee cho BTC / ETH / BNB / SOL / XRP
- Hero bài nổi bật + cụm bài phụ dạng magazine
- Layout main content 70/30 với sidebar widget
- Tin mới nhất dạng grid 2 cột
- Phân tích thị trường dạng list
- Banner BingX tông dark gold
- Footer 3 cột theo brief
- Responsive mobile-first, không dùng framework

## Chạy local

```bash
cd /home/shinyyume/.openclaw/workspace/signal-hunters-web
python3 -m http.server 8080
```

Mở `http://localhost:8080`
